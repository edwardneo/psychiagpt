DSM.csv is a dataset generated from the anxiety and depression sections of the DSM 5. 
First, the section PDf files were downloaded. Next, they were converted to .txt
Then, they were preprocessed using the .ipynb into DSM.csv.
DSM.csv has 1919 rows x 2 columns. The left column is for sentences, and the right column is a binary label (1 for anxiety, 0 for depression)
The first 1123 rows are for anxiety, the next 796 rows are for depression.
